# Trisha Portfolio Starter

This is a basic portfolio website made with plain HTML, CSS, and JavaScript.

## Files

- `index.html` = page structure and content
- `style.css` = colors, layout, spacing, and responsive design
- `script.js` = mobile menu and card hover effect

## How to view it

Double-click `index.html` to open it in your browser.

## How to edit it

Open the folder in VS Code.

Change text in `index.html`.
Change colors/layout in `style.css`.

## What to customize first

1. Replace project descriptions with your real projects.
2. Add your real LinkedIn link.
3. Add a resume PDF later if you want.
4. Add screenshots/images in an `images` folder later.
